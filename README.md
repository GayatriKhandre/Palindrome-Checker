# Palindrome Checker (Python)

This is a simple Python program that checks if the text you enter is a **palindrome**.  
A palindrome is a word or sentence that reads the same forwards and backwards (ignoring spaces, punctuation, and capital letters).

---

## Features

- Checks if a word or sentence is a palindrome
- Ignores case, spaces, and special characters
- Easy to use from the terminal

---

## Requirements

- Python 3  
(No extra libraries are needed)

---

## How to Run

1. Download or clone this repository.
2. Open a terminal and navigate to the project folder.
3. Run the script:

```bash
python palindrome_checker.py
